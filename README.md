# Pong - COMP7051 Assignment 1
## Controls
Player 1
```
Move Up: w
Move Down: s
```

Player 2
```
Move Up: UpArrow
Move Down: DownArrow
```

## Console Commands
```
// Change color of background
background=red
background=blue
background=cyan
background=gray
background=green
background=megenta
background=yellow

// Change ball size
ball=big
ball=small
```
