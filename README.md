# Pong - COMP7051 Assignment 1
## Controls
Player 1
```
Move Up: w / Up on JoyStick 1
Move Down: s / Up on JoyStick 1
```

Player 2
```
Move Up: UpArrow / Up on JoyStick 2
Move Down: DownArrow / Up on JoyStick 2
```

## Console Commands
```
// Change color of background
background=red
background=blue
background=cyan
background=gray
background=green
background=megenta
background=yellow

// Change ball size
ball=big
ball=small
```
